Selected Projects

Robust Loss Functions

Problem
Standard cross-entropy degrades sharply when training labels are noisy.
Approach
Implemented CE, FL, NCE, NFL, and APL losses and compared them on CNNs built from scratch, trained on CIFAR-10 with up to 40% injected label noise.
Outcome
Systematic evaluation of loss robustness across noise rates, written up in a technical report.
PyTorchCIFAR-10CNNs
code + report

Diffusion from the Ground Up

Problem
Understand what actually drives sample quality in denoising diffusion models.
Approach
Trained a DDPM on CIFAR-10 and ablated classifier-free guidance, sampling steps, and attention backbones (DiT vs. Xformers).
Outcome
Generation quality quantified with FID and CMMD across configurations.
DDPMDiTFID / CMMD
code + report

Neural Net from Scratch

Problem
Backpropagation is easy to use and easy to misunderstand — so derive and build it by hand.
Approach
Fully-connected network in pure NumPy: forward pass, backprop, and SGD written from first principles, trained on MNIST.
Outcome
97.8% test accuracy with no ML framework involved.
